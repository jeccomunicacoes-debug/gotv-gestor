🔧 Configuração Firebase
Para sincronizar dados entre o PC e o celular, precisas configurar o Firebase (gratuito). Segue os passos abaixo — demora cerca de 5 minutos.

1 Criar projecto no Firebase

1. Vai a console.firebase.google.com e entra com o teu Gmail
2. Clica em Criar projecto
3. Dá um nome: gotv-gestor
4. Desactiva o Google Analytics (não precisas) → Criar projecto

2 Activar Realtime Database

1. No menu lateral clica em Build → Realtime Database
2. Clica Create Database
3. Escolhe localização: europe-west1 (mais próximo)
4. Selecciona Start in test mode → Enable

3 Obter as credenciais

1. Clica no ícone de engrenagem ⚙️ → Project settings
2. Em Your apps clica no ícone </> (Web app)
3. Dá um nome: gestor → Register app
4. Vai aparecer um bloco de código com os valores abaixo — copia cada um

4 Guardar e activar

Após guardar, o ficheiro HTML é actualizado com as tuas credenciais. Abre-o no browser, faz upload para o Google Drive e acede a partir de qualquer dispositivo com o mesmo link.
